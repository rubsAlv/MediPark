En este repositorio se incluyen los modelos entrenados en formato .pkl, el código .py utilizado para su entrenamiento y el módulo que integra estos modelos en la API. Solo se suben estos componentes porque son los elementos necesarios para la aplicación móvil y la versión de Wear OS; las demás partes del proyecto corresponden a las interfaces específicas de cada plataforma y no forman parte de este repositorio.
